# AI-Powered Stock Screener (Sprint 1)

## Overview
This project is an AI-powered mobile stock screener platform that allows users to query stock data using natural language. Sprint 1 focuses on system foundation and backend setup.

## Completed in Sprint 1
- Requirements & field catalog finalized
- PostgreSQL database schema designed
- Integrated Alpha Vantage market data API
- Backend API gateway using Node.js & Express
- JWT-based authentication setup
- Basic CI/CD pipeline

## Tech Stack
- Backend: Node.js, Express
- Database: PostgreSQL
- Market Data: Alpha Vantage
- Authentication: JWT
- CI/CD: GitHub Actions

## Next Steps (Sprint 2)
- LLM-based natural language parser
- Screener compiler and SQL runner
- Mobile UI prototype
