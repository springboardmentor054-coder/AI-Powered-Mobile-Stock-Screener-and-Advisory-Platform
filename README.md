# AI-Powered-Mobile-Stock-Screener-and-Advisory-Platform

## 📖 Project Description
This project is an AI-powered mobile stock screener and advisory platform designed to help users analyze stocks and make informed investment decisions.  
The system uses structured financial data and stock market information to screen companies based on various parameters.

This project is developed as part of my **Infosys Virtual Internship**.

---

## 🎯 Objectives
- To store and manage stock market data efficiently
- To screen stocks based on financial parameters
- To support decision-making using structured data
- To understand database design and integration in real-world projects

---

## 🛠️ Technologies Used
- **Backend:** Node.js, Express.js
- **Database:** SQLite / MySQL (as applicable)
- **Tools:** VS Code, Git, GitHub
- **API:** REST APIs (for data access)

---

## 🗄️ Database Structure
The database includes the following tables:
- **Companies** – Stores company details
- **Stocks** – Stores stock price and market data
- **Financials** – Stores financial statements and ratios

---

## ⚙️ Features
- View company stock data
- Retrieve financial information
- API-based data access
- Clean and structured database design
